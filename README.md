# JPhoto

Photo-club with a powerful functionality and admin console.   
Legacy and is not supported anymore.  

User with ID #1 is a site admin by default.

## Run container with DB
```shell
cd jphoto
./docker-compose.yml 
```

## Run application
```shell
cd jphoto
./deploy-local.sh
```
